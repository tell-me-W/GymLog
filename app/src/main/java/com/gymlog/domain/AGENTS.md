# GymLog Domain Guide

`domain` 영역은 UI나 Room에 묶이지 않는 순수 계산과 정책을 담당합니다. 입력과 출력이 명확한 함수로 유지하고 JVM 테스트로 보호합니다.

## Responsibilities

- 총 볼륨 계산.
- 월간/세션 요약 계산.
- 세트 추가/삭제 가능 조건.
- 세트 값 전파 대상 판단.
- 저장 전 데이터 정리 정책 중 순수 계산으로 분리 가능한 부분.

## Domain Rules

- 총 볼륨은 완료 체크된 반복/중량 세트의 `중량 kg * 반복 횟수` 합계입니다.
- 시간 기준 운동 세트는 수행 기록에는 남기지만 총 볼륨에는 포함하지 않습니다.
- 월간 요약은 선택 월의 `COMPLETED` 세션과 완료 세트만 기준으로 계산합니다.
- 한 세트의 `kg` 또는 `reps` 수정은 같은 운동 종목 안의 미완료 세트에만 전파합니다.
- 이미 완료한 세트와 다른 운동 종목의 세트는 값 전파 대상이 아닙니다.
- 새 세트는 마지막 세트의 값을 기본값으로 사용합니다.
- 세트 삭제는 마지막 세트만 대상으로 하며 한 운동에는 최소 1개 세트를 유지합니다.

## Testing Focus

- 총 볼륨 계산.
- 시간 기준 운동의 볼륨 제외.
- 월간 요약 계산.
- 완료/미완료 세트 필터링.
- 같은 운동 안의 미완료 세트 값 전파.
- 마지막 세트 복사 추가와 삭제 가능 조건.
