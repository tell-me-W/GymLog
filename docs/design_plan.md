# GymLog UI/UX 디자인 개편 계획 (Design Revamp Plan)

이 문서는 제공된 디자인 시안(image 1~5)을 바탕으로 기존 GymLog 앱의 UI를 개선하기 위한 작업 계획입니다. Jetpack Compose를 사용하여 각 화면과 컴포넌트를 시안과 동일한 수준의 세련된 디자인으로 구현합니다.

## User Review Required

- **에셋(Asset) 관련**: 폰트, 근육 해부도 이미지, 네비게이션 아이콘 등의 리소스는 현재 프로젝트에 없으므로, 기본 Material 아이콘과 Placeholder Box(도형)로 레이아웃과 형태를 우선 구현합니다. 실제 리소스가 있다면 추가가 필요합니다.
- **디자인 폰트**: 시안에 있는 독특한 폰트는 커스텀 폰트(TTF/OTF) 파일이 필요합니다. 이번 개편에서는 구조와 레이아웃을 우선 맞추고 폰트는 시스템 폰트로 둡니다. 필요한 경우 폰트 파일을 제공해주시면 반영 가능합니다.
- **PRO 기능 더미 처리**: 시안에 있는 PRO 배너나 PRO 버튼은 현재 백엔드/결제 기능이 없으므로 UI로만 표시(Dummy)됩니다.

## Proposed Changes

### 공통 컴포넌트 및 테마 (Theme & Components)
- **Colors**: 다크 배경 색상(`Color(0xFF121212)` 등), 블루 포인트 컬러, 텍스트 컬러 토큰을 정의합니다.
- **BottomSheet**: 기존 `AlertDialog`로 처리되던 종목 추가, 기록 불러오기 다이얼로그를 Compose의 `ModalBottomSheet`로 변경합니다.
- **Buttons & Chips**: 모서리가 둥근 버튼 및 필터 칩(가로 스크롤 지원) 컴포넌트를 생성합니다.

### 1. 대시보드 화면 (DashboardScreen) - [image5 참조]
- **배경 및 캘린더**:
  - 화면 상단을 다크 모드 스타일로 구성.
  - 연/월 헤더 좌측 정렬 및 `<` `>` 버튼 분리.
  - 날짜 선택/완료 시 파란색 둥근 배경 적용.

### 2. 종목 추가 바텀 시트 (AddExerciseBottomSheet) - [image1, 3, 4 참조]
- **기존 `AddExerciseDialog`를 대체**
- **상단바 & 검색**: 헤더 레이아웃 구성, 둥근 회색 배경의 검색 바(TextField) 추가.
- **필터 칩 (LazyRow)**: '전체', '가슴', '등' 등의 부위를 가로 스크롤로 제공.
- **종목 리스트 UI**:
  - 좌측에 둥근 사각형 이미지, 우측 하단에 미니 인체 해부도 Placeholder 겹침 (Box).

### 3. 이전 기록 불러오기 바텀 시트 (CopyFromDateBottomSheet) - [image2 참조]
- **기존 `CopyFromDateScreen`을 바텀 시트로 대체**
- **날짜 탭 (LazyRow)**: 가로로 넘길 수 있는 날짜별 운동 요약 칩 제공.
- **운동 요약 정보 (Summary)**: 
  - "3종목 • 27분" 텍스트.
  - 각 부위별 인체 해부도 큼직한 이미지(Placeholder)와 세트수 표시 뷰.
- **운동 상세 리스트**: 종목 이름 아래에 "4세트 • 45kg • 15회" 요약 표시.
- **하단 액션 버튼**: '빈운동' (다크 그레이) / '불러오기 PRO' (블루) 버튼 나란히 배치.

## Verification Plan

### Manual Verification
1. **대시보드 확인**: 앱 실행 시 다크 테마 캘린더, 네비게이션 바가 시안(image5)과 일치하게 나타나는지 확인합니다.
2. **종목 추가 확인**: 대시보드에서 '자유 운동' 또는 운동 추가 시 바텀 시트가 부드럽게 올라오며, 필터 가로 스크롤과 리스트 항목 디자인이 시안(image1)과 맞는지 확인합니다.
3. **불러오기 확인**: 이전 기록 불러오기 기능 실행 시 상단 날짜 탭, 중간 요약 인체 이미지, 하단 두 개의 버튼 구조가 시안(image2)과 맞는지 확인합니다.
