# Exercise Library And Routines

운동 라이브러리는 기본 seed 운동, 커스텀 운동, 최근 수행 기록 정렬, 루틴 저장을 담당합니다.

## Exercise Picker

- 운동 종목 추가 화면은 여러 종목을 한 번에 선택할 수 있어야 합니다.
- 운동명 검색을 지원합니다.
- 타겟 부위 필터에는 `전체`, `가슴`, `등`, `하체`, `어깨`, `팔`, `복근`, `기타`, `유산소`가 있습니다.
- `전체`는 모든 운동을 보여주는 필터 없음 상태입니다.
- `기타`는 실제 필터로 동작합니다.
- 최근 2개월 수행 기록이 있는 종목은 최신순으로 우선 정렬합니다.
- 기록이 없는 종목은 뒤쪽에 이름순으로 정렬합니다.
- 최근 수행 기록은 `최근 yyyy.MM.dd · N세트` 형식으로 표시합니다.

## Custom Exercises

- 커스텀 종목 추가 폼은 기본으로 접혀 있어야 하며 사용자가 열고 닫을 수 있어야 합니다.
- 커스텀 종목은 이름, 타겟 부위, 기본 휴식 시간을 지정해 추가합니다.
- 커스텀 종목의 분류에는 `전체`를 저장하지 않습니다.
- `기타 횟수 기준 운동`은 `기타` 분류의 반복/중량 기준 운동입니다.
- `기타 시간 기준 운동`은 `유산소` 분류의 시간 기준 운동입니다.

## Seed Exercises

- 앱 시작 시 기본 운동 seed를 DB와 동기화합니다.
- 같은 이름의 기본 종목이 없으면 seed 기준으로 추가합니다.
- 같은 이름의 기본 종목이 `기타` 커스텀으로 들어간 경우 seed 기준 타겟 부위, 휴식 시간, 기본 종목 여부로 보정합니다.
- 가져오기 중 기본 운동명이 나오면 `SeedExercises`의 타겟 부위와 휴식 시간을 우선 사용합니다.

## Routines

- 루틴은 자주 함께 수행하는 운동 종목 목록을 저장하는 기능입니다.
- 루틴은 종목 목록과 순서만 저장합니다.
- 세트 수, 중량, 반복 횟수, 시간 값은 루틴에 저장하지 않습니다.
- 저장된 루틴을 누르면 현재 드래프트 운동에 루틴 종목들이 순서대로 추가됩니다.
- 루틴으로 추가된 각 운동은 기본 1세트로 시작합니다.
- 저장된 루틴은 삭제할 수 있습니다.
