# Demo Video Descriptions

## 01_full_workout_logging_flow.mp4

 - 대시보드
 - 운동 시작 (빈 운동으로 시작)
 - 종목 여러개 추가
 - 세트(kg-횟수) 입력 (완료전 세트 자동 연동)
 - 세트 완료 후 휴식 타이머 시작
 - 종목 삭제
 - 루틴 입력 
 - 휴식 끝나면 알람 
 - 알람을 누르면 앱 화면으로 복귀
 - 운동 완료 - 기록 공유
 - 대시보드 월간 요약 반영
 - 기록 확인 및 삭제
