# Workout Logging

운동 로깅 흐름은 드래프트 우선 저장, 이전 기록 복사, 완료 체크 기준 저장을 따른다.

## Starting Workouts

- `빈 운동으로 시작`은 새 `DRAFT` 세션을 만들고 로깅 화면으로 이동합니다.
- 이전 기록 복사는 `COMPLETED` 세션만 원본으로 허용합니다.
- 복사 결과는 새 `DRAFT` 세션이어야 합니다.
- 복사된 모든 세트의 `isCompleted`는 `false`로 초기화합니다.
- 기존 드래프트 세션은 새 운동 시작이나 기록 복사 때문에 자동 삭제하지 않습니다.

## Live Logging

- 운동 중 세트 추가, 중량, 반복 수, 완료 체크 변경은 드래프트에 즉시 저장합니다.
- 운동 종목 카드는 운동명, 타겟 부위, 기본 휴식 시간을 보여줍니다.
- 세트 추가는 마지막 세트의 중량/반복 수 또는 시간 값을 기본값으로 사용합니다.
- 세트 삭제는 마지막 세트만 대상으로 하며, 1번 세트만 남으면 삭제하지 않습니다.
- 반복/중량 기준 운동은 `kg`, `reps`, 완료 체크를 입력합니다.
- 시간 기준 운동은 분 단위 시간과 완료 체크를 입력합니다.
- 한 세트의 `kg` 또는 `reps` 수정은 같은 운동 종목 안의 미완료 세트에만 전파합니다.
- 이미 완료한 세트와 다른 운동 종목의 세트는 값 전파 대상이 아닙니다.

## Rest Timer

- 세트 완료 체크 시 해당 운동의 기본 휴식 시간을 기준으로 휴식 타이머가 시작됩니다.
- 휴식 타이머 알림은 휴식 종료 시점에만 사용자 알림을 울립니다.
- Android 13 이상에서는 휴식 타이머 완료 알림을 위해 `POST_NOTIFICATIONS` 권한을 요청합니다.

## Completing Workouts

- `운동 완료`를 누르면 완료 체크된 세트만 저장 대상입니다.
- 완료 체크되지 않은 세트는 삭제합니다.
- 완료 체크된 세트가 하나도 없는 운동 종목은 기록에서 제외합니다.
- 세션 전체에 완료 세트가 하나도 없으면 `COMPLETED`로 저장하지 않고 드래프트를 삭제합니다.
- 저장된 기록의 총 볼륨은 반복/중량 기준 세트의 `중량 kg * 반복 횟수` 합계입니다.
- 시간 기준 운동 세트는 수행 기록으로 저장하지만 총 볼륨에는 포함하지 않습니다.

## Sharing

- 운동 완료 화면에서는 완료한 운동 요약을 공유할 수 있습니다.
- 텍스트 공유는 날짜, 운동 시간, 총 볼륨, 종목 수, 세트 수를 Android 공유 시트로 보냅니다.
- 이미지 공유는 앱 내부에서 인증 카드 PNG를 생성해 Android 공유 시트를 엽니다.
- 공유 파일은 앱 cache 디렉터리에 생성하며 원본 운동 기록은 변경하지 않습니다.
